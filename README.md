# netflix_clone_ui
# 🎬 Netflix Clone UI

A pixel-perfect Netflix clone built with pure HTML, CSS, and JavaScript — featuring a fully responsive design, modern dark UI, and smooth animations.

---

## ✨ Features

- 🎨 Netflix-style dark UI with red accent theme
- 📱 Fully responsive — works on mobile, tablet & desktop
- 🎞️ Movie/show cards with hover effects
- 🔥 Featured hero banner section
- 📂 Category rows (Trending, Top Rated, etc.)
- 🔍 Search bar UI
- 🧭 Sticky navbar with scroll effect
- ⚡ Smooth CSS animations & transitions
- 🖱️ Horizontal scroll for movie rows

---

## 🛠️ Tech Stack

| Technology      :   Usage 
|----------------------------------------------------|
| HTML5           :  Structure & layout 
| CSS3            :  Styling, animations, responsive design 
| JavaScript      :  Interactivity & DOM manipulation 
| Font Awesome    :  Icons 
| Google Fonts    :  Typography
